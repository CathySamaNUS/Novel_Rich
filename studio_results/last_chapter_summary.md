# 连续性记录更新

## 一、本章发生的大事

1. **沈栀与宋砚在旧书廊见面**：宋砚提前为沈栀标注了民国酒店发展史的重点段落，展现周到体贴的一面。两人之间产生微妙的身体接触（递书时指尖相碰，停顿约三秒）。

2. **沈栀发现可疑线索**：
   - 在旧书廊二楼偶遇一名穿深灰西装的中年男人打电话，内容涉及“陆先生那边的文件，沈家酒店那块地”。
   - 在旧书中发现一张泛黄老照片，照片角落有轮廓近似年轻时的陆骁父亲的人物。

3. **陆骁的“警告性示好”**：陆骁通过私人司机递送一份沈家正在接触的三家酒店项目资料，右上角写“仅供参考”，暗示他清楚沈栀的行踪和谈判内容。

4. **顾宁主动介入**：顾宁通过陌生号码发来短信，询问沈栀是否想聊沈家国内项目的事，表明她已知沈栀、宋砚见面，并打算亲自下场。

5. **宋砚与顾宁的旧联结被证实**：
   - 顾宁电话中提到宋砚曾带她去过同一家旧书廊“几次”。
   - 宋砚承认“以前陪顾宁来过几次”，原因是“那时候陆骁不在，有些场合她一个人不好应付”。
   - 宋砚解释其帮助顾宁的动机：“一开始是陆骁提过一句。后来……我自己觉得应该帮。”

## 五、下一章必须记住的信息

1. **沈栀当前持有的关键物品**：
   - 蓝宝石胸针（陆骁所赠，放于包内袋）
   - 民国酒店发展史旧书（宋砚所赠，带标注）
   - 三份酒店项目资料（陆骁通过司机送达）
   - 顾宁的约谈短信

2. **沈栀的决策困境**：
   - 面对宋砚的温柔和陆骁的掌控，她心里对两人的天平在微妙地摇摆
   - 沈家项目的棋局上，三个人（陆骁、宋砚、顾宁）都已伸手，每个人都在用自己的方式靠近她
   - 她既不想被陆骁推着走，也不想被宋砚“温柔地安排”，但尚未找到自主权的支点

3. **必须确认的设定一致性**：
   - 宋砚与顾宁有旧：确认，且原因是“陆骁不在时，宋砚陪顾宁应付场面”
   - 陆骁对沈家项目的介入深度：比宋砚更早、更深，且不公开动机
   - 顾宁知晓沈栀行踪的方式：圈内信息互通（可能有第三方）
   - 故事时间线：沈栀回国后，已陆续接触宋砚（三次）、陆骁（多次）、顾宁（两次）

4. **下一章最大钩子**：
   - **顾宁的约**：沈栀会赴约吗？如果去，顾宁会以什么姿态出场？她会拿出什么筹码？
   - **陆骁的“资料”**：沈栀下一步会利用这份资料验证什么信息？还是会发现它和宋砚标注的书籍之间存在矛盾？
   - **照片的悬念**：旧照片中疑似陆骁父亲的人物——这本书廊的“老人聚会”究竟和陆家有什么历史渊源？

5. **情绪记忆节点**：
   - 沈栀在书房门口“忘了呼吸”的三秒：她对宋砚确实有心动，不是完全理性的
   - 沈栀合上资料后“手心发凉”：陆骁的高强度掌控让她生理上感到压迫
   - 沈栀在夜色中翻开旧书的第一个标注：她选择自己看，不等待任何人替她解读——这是她作为主角第一次主动介入信息层

---

**总结：本章是转折之章。**
沈栀从“被动接受接近”转变为“开始主动审视信息”。旧书廊见面完成了三个关键任务：①深化宋砚线（好感和警示同时出现）；②揭示陆骁线（掌控力升级，动机成谜）；③激活顾宁线（从旁观者变直接参与者）。棋桌上四人全部露面，齿轮已经开始转动。
